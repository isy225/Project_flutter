# Assessment 01: User Profile Screen - UNLOX

## 📌 Project Overview
This project is a static User Profile interface developed as part of the **UNLOX** assessment. The primary goal was to demonstrate proficiency in Flutter layout construction using exclusively **Stateless Widgets**.

## 📸 App Preview (UI Mockup)
![User Profile Preview](https://raw.githubusercontent.com/isy225/Project_flutter/master/user_profile_ui/screemshots/app_preview.png)
*(The interface is designed to be clean, readable, and professional, following the RIVOQUIX LEARNING guidelines)*

## 🛠️ Technical Requirements Met
According to the official assessment criteria:
*   **Architecture**: Built 100% with **Stateless Widgets** to manage a static UI efficiently.
*   **Widgets Implemented**: 
    *   `Scaffold` & `AppBar` for the core app structure.
    *   `CircleAvatar` for the profile image at the top.
    *   `Column` & `Row` for precise element alignment.
    *   `Text` with custom styles (bold headings for name, centered bio).
    *   `Container` & `Padding` for professional spacing and layout.
    *   `ElevatedButton` for the "Follow" action button.
*   **Asset Management**: Local profile image stored in the `assets/` directory and correctly declared in `pubspec.yaml`.

## 📂 Repository Structure
*   `lib/main.dart`: Main source code containing the UI logic.
*   `assets/`: Local image assets used in the application.
*   `screemshots/`: Screenshot of the final rendered application.

## 🚀 How to Run the Project
1. Clone the repository: `git clone <your-github-link>`
2. Get dependencies: `flutter pub get`
3. Run the application: `flutter run`
